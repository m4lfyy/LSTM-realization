## Реализация LSTM с помощью keras на примере прогнозирования цен на акции

Для этой задачи использовались данные из *Yahoo Finance*, а конкретно информация о торгах золота в период с **2015** по **2020**  год. 

### Параметры запуска:
- Python 3.8.12
- numpy 1.24.3
- pandas 1.4.2
- keras 2.9.0
- sklearn 1.2.0
- matplotlib 3.5.3
